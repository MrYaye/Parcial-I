# Brayan Stiven Zamora Arias

## Descripción

Se desarrollo una landing page inspirada en el sitio de la pagina de ExpressVPN utilizando **HTML5 y CSS3**. El objetivo fue recrear la estructura visual de la página aplicando lo utilizado en clases y herramientas conocidas.


## Estructura del proyecto

* `index.html` = estructura HTML de la página
* `style.css` → estilos y diseño
* `image/` → Imagenes utilizadas
